YNAB on the web

Ever wanted to check your YNAB data on the web? Look no further! With this application you can view and filter your transaction log if you have it synced with Dropbox.

Sign in with Dropbox

How does this work?

If you are syncing your YNAB data with Dropbox, then by giving this application the permission to access your data on Dropbox, it can read the same data that the official YNAB application uses.

Is this safe?

This app uses the Dropbox JavaScript SDK, so all the communication happens in the browser, which means none of your data goes through this server.
